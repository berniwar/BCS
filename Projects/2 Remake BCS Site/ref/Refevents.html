This weekend workshop is designed

Event Details

This weekend workshop is designed for people interested in learning about React.js.

React.js is an open-source JavaScript library for building user interfaces created by Facebook.

It aims primarily to provide speed, simplicity and scalability. React processes only user interfaces in applications. It is intended for the single-page application sending and getting data without refreshing a page with modular approach.

During this workshop we are going to build a ToDo application with React.

Daily schedule:

10:00-13:00: classwork

13:00-14:00: a break

14:00-17:00: classwork

THIS COURSE IS IN ENGLISH.

YOU WILL NEED TO BRING YOUR LAPTOP.

By the end of this course you will learn:

  • React logic and modular approach;
  • JSX syntax;
  • React components;
  • Pass data using JSX;
  • State;
  • Node environment;
  • Working with Git and GitHub;
  • React debugging.

You will build a ToDo app working in the browser with React.js

Event Details

Barcelona Code School is happy

Event Details

Barcelona Code School is happy to host a free coding workshop for women, members of the LGBTQ community and for people from underrepresented groups in the tech industry. We are glad to have a new friend and partner and to support this honorable project.

The event is held by Codebar, a non-profit initiative that facilitates the growth of a diverse tech community by running regular programming workshops. It originally started in the UK, but now it has also a chapter in Barcelona. The workshops are a great way for students to get help with coding in a safe welcoming place, but it is also a great way for them to informally be mentored by working developers.

It is for the students of all the levels and total beginners are very welcome.

Sign up

FORMAT OF THE WORKSHOP

18:30-19:00 People get to know each other over drinks and food.
19:00  A presentation from the Barcelona Code School
19:15  We pair students and coaches and begin to work
21:00 The workshop ends.

FREQUENTLY ASKED QUESTIONS

ATTEND CODEBAR AS A STUDENT. IS THERE ANYTHING YOU SHOULD DO BEFORE SIGNING UP?

Codebar is about getting people excited about programming. If you are an absolute beginner, feel free to join us. We will help you learn some basic coding skills as well as help you explore more resources. If you are signing up as a student, we suggest that you bring a laptop as it will be easier for you to carry on working on things after the workshops. If you don’t have one, send us a message so we can try and help out. Make sure that you RSVP either through the meetup page or on the link.

ATTEND CODEBAR AS A COACH. IS THERE ANYTHING YOU SHOULD DO BESIDES SIGNING UP?

Attending codebar as a coach is very easy and a lot of fun. Just make sure that you sign up either through the meetup page or athttps://codebar.io/barcelona. It would be great if you can read our coaching guide and if you have time, get familiar with the tutorials.

The tutorials are open source so if there is something that you want to improve open an issue on the GitHub repository or send a pull request. We don’t expect the coaches to be familiar with all the technologies and languages we have material on.

(Thursday) 6:30 pm - 9:00 pm CET

A website is your online

Event Details

A website is your online business card, the first thing your customers see if they are trying to get in touch. Doesn’t matter if your business is big or small, if you are just trying to start some new project or improve your online presence – you do need a website.

And since currently 41% internet users are surfing from laptops and desktops, 54% are using mobile devices and 5% are using tablets you need to make sure that your website is responsive, i.e. looks and works same good on any of user devices.

This weekend intensive course will give you a solid hands-on introduction to HTML5 and CSS3 applied to the goals of creating a modern and responsive websites. We are going to work on 2 projects: a single page website and multipage website.

By the end of this course you will be able to:

  • Create responsive websites with navigation on the same page or several pages
  • Understand what responsiveness means and how to apply it
  • Using CSS native Flexbox layout mode to prepare your website for different user devices
  • Using Bootstrap for fast building responsive layout with grid
  • Use Chrome Dev Tools for browser-based web development
  • Have 2 finished responsive websites: a single page and a multipage
  • Publish them to GitHub and have a shareable link with GitHub Pages

This after-school course is designed to introduce our youngest students to the programming with JavaScript. We are going to explore logic behind programming, algorithmic approach to solving problems, JavaScript basics and explore how we can use JavaScript to build working applications.

Along the way we will build several websites and see how JavaScript can convert them to applications – not just to display static content but also to act and respond to users actions.

During the course we are going to learn:

  • Logic behind programming language
  • JavaScript syntax
  • Variables, functions, loops, arrays, objects, conditionals
  • HTML
  • CSS
  • Game logic

The course starts on September 18 and ends on June 18 and is taught in English.
